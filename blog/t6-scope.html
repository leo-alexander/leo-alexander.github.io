Scope

In JavaScript, variables can have one of two scopes: global or local. Global variables are declared outside a function, and can be accessed from anywhere in the code. A variable decrlared within a function is local, and can only be accessed from within that function (and nested functions - also called nested scope).

Heres a simple example: 

    var glob = "Global Scope";

    function local() {
      var loc = "Local Scope";
    }

  console.log(glob); //"Global Scope"
  console.log(loc); //"loc is not defined"
