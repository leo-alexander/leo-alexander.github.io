Javascript Basics

This week marked the start of learning Javascript. While HTML and CSS determine what is displayed on a web page, Javascript adds another layer, allowing the web page responds to events that happen. Without JS, the web would look great but would be nowhere near as useful. Google, Facebook, Youtube, Wikipedia and most other major websites all use JavaScript as their front-end language.

Two ways of handling data in JS are "Arrays[]" and Objects {}:

Array:var myGarden = ["pohutakawa", "mount cook lily", "kowhai", "lavender"]

Object:

var myFlower = {
    "name": "Mount Cook Lily",
    "latin": "Ranunculus lyallii",
    "location discovered": ["South Island", "Stewart Island"]
  };
}

Accessing data from the above array would involve var myGarden[1] which would return "mount cook lily".

Accessing data from the above object (ie. properties) would involve using the dot operator (when you know the name of the property you're trying to access): myFlower.name; or using bracket notation: myFlower["location discovered"];

Functions in JS are blocks of code that perform certain tasks. They are useful because they organise tasks into seperate packages, so can be executed or halted without other code being effected.
